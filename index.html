Web Technologies I Know

HTMl 5

HTML, or HyperText Markup Language, is the standard markup language for creating web pages. It is used to define the structure and content of a web page.

CSS 3

CSS, or Cascading Style Sheets, is a language used to style the appearance of web pages. It is used to control the layout, fonts, colors, and other aspects of a web page's appearance.

JavaScript

JavaScript is a scripting language used to add interactivity to web pages. It is used to create dynamic effects, such as menus, animations, and games.

ReactJS

React is a JavaScript library used to build user interfaces. It is known for its speed, scalability, and ease of use.
